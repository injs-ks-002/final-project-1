# Final Project 1
## Pre-Requisite

Fill in your database configuration first in the `/config/db.js` directory

Example:
```sh
user: '', // your database username
host: 'localhost',
database: '', // your database name
password: '', // password of your database username 
port: '5432'
```

## Install

```sh
$ npm install
```

Then, at the command prompt, type
```sh
$ npm run db:migrate
```

## Usage

```sh
$ npm start
```
